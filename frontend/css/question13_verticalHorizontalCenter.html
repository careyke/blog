# 2.垂直居中

# 子元素的行内元素（inline,inline-block）

  1. 父元素有确定的高度，使用line-height + vertical-align
.father{
  height: 100px;
  line-height: 100px;
}
.son{
  vertical-align: middle;
}
1
2
3
4
5
6
7

如果子元素是inline元素，不需要设置vertical-align，因为inline元素在行框中垂直居中。如果子元素是inline-block，就需要设置vertical-align: middle。 2. 使用vertical-align + 伪元素，父元素宽度确不确定都可以。

.father::after{
  display: inline-block;
  content: '';
  height: 100%;
  vertical-align: middle;
}
.son{
  vertical-align: middle
}
